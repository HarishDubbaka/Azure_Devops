# 📦 Azure Repos

Azure Repos is a set of version control tools that you can use to manage your code. Whether your software project is large or small, using version control as soon as possible is a best practice.

## 📘 What is Version Control?

Version control systems help you track changes in your code over time. As you edit your code, the system captures snapshots of your files. These snapshots are stored permanently, allowing you to:

* 🔍 Review history
* 🔄 Roll back to previous versions
* 🤝 Collaborate with teams

Even as a single developer, version control helps you stay organized while fixing bugs and developing new features.

---

## 🔀 Types of Version Control in Azure DevOps

Azure DevOps supports two source control options:

### 🧩 Git (Recommended)

Git is a **distributed version control system**, meaning each developer has a local copy of the repository.

#### 🔑 Key Capabilities:

* 📄 Review files and history
* 💻 Download and edit locally
* 📌 Track commits
* 🔁 Use pull requests for collaboration
* 🏷️ Create tags for version marking

---

### 🏢 TFVC (Team Foundation Version Control)

TFVC is a **centralized version control system** where all code is stored on a central server.

#### 🔑 Key Features:

* 📂 Single version workflow
* 🗄️ Server-side history management
* 🌿 Path-based branching

---

## 🧰 Development Environment Integration

Azure Repos supports a wide range of tools and platforms:

* 🌐 Cross-platform: Android, iOS, Linux, macOS, Windows
* 🧑‍💻 IDEs: Visual Studio, VS Code, IntelliJ, Eclipse, Android Studio, Xcode
* 💻 Languages: .NET, Java, Node.js, Python, PHP, Ruby, and more
* 🔄 Flexible clients: Git & TFVC support

# ⚙️ Azure Pipelines

Azure Pipelines is a service that enables **Continuous Integration (CI)**, **Continuous Testing**, and **Continuous Delivery (CD)**.

It automates the process of:

* 🏗️ Build
* 🧪 Test
* 🚀 Deploy

---

## 🎯 Benefits of Azure Pipelines

* 🔄 Deploy to multiple targets simultaneously
* ☁️ Seamless Azure integration
* 🔗 GitHub integration
* 🌍 Supports any language/platform
* 💻 Works on Windows, Linux, macOS
* 🆓 Supports open-source projects

---

## 📌 Prerequisites

To use Azure Pipelines, you need:

* An Azure DevOps organization
* Source code in a version control system

---

## 🧑‍💻 Supported Languages & Platforms

Azure Pipelines supports:

* Node.js, Python, Java, PHP, Ruby
* C#, C++, Go, .NET
* Android & iOS applications

You can run builds in parallel on:

* 🐧 Linux
* 🍎 macOS
* 🪟 Windows
